# Premium Page - 글로벌 전자 카탈로그 SaaS 플랫폼 전략

## 1. 프로젝트 개요
`Premium Page`는 기존의 `jcatalog` 프로젝트와 `premiumpage` 프로젝트를 통합하여 탄생한 **글로벌 전자 카탈로그 전문 SaaS 플랫폼**입니다. 
- **jcatalog**: 하이엔드 3D 인터랙티브 브로슈어 제작 역량 (Flagship: EMT GLOBAL TECH)
- **premiumpage**: 확장 가능한 SaaS 아키텍처, 견적 시스템, 템플릿 라이브러리 기반

## 2. 핵심 비즈니스 모델 및 서비스 라인업

### A. PDF-to-Web 변환 서비스 (Subscription / Lite)
- **대상**: 기존에 제작된 PDF 리플렛, 브로슈어를 가진 모든 기업
- **기능**: PDF 업로드 시 웹 기반 페이지 플리핑(Flip) 효과가 적용된 전자 카탈로그로 즉시 변환
- **강점**: 뷰어 설치 불필요, QR/링크 하나로 글로벌 배포 가능, 낮은 월 구독료

### B. 프리미엄 커스텀 제작 서비스 (Agency / Standard)
- **대상**: 해외 수출용 전문 카탈로그가 필요한 제조/기술 기업
- **기능**: 반응형 레이아웃, 제품 검색, 다국어 지원, 문의 폼 연동이 포함된 전문 전자 카탈로그 기획 및 제작

### C. 마스터피스 3D 에디션 (Masterpiece / High-end)
- **대상**: 브랜드 가치 극대화와 압도적 시뮬레이션이 필요한 하이엔드 기업
- **기능**: `Three.js` 기반 3D 모델링, 스크롤 인터랙션, EMT 스타일의 고급 애니메이션 (Flagship Piece 활용)

## 3. 기술 스택 및 SaaS 전략
- **Frontend**: Next.js 16 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS 4, Framer Motion (고급 애니메이션), Lucide Icons
- **Backend & Database**: Prisma with SQLite/PostgreSQL
- **Global Identity**: 'Premium Page'라는 단일 브랜드로 글로벌 마켓 타겟팅
