# EMT 웹사이트 수정 작업 로그 (2026-01-21)

## 1. 작업 개요

- **목표**: 웹사이트 콘텐츠의 정확성 향상 및 디자인 통일성 확보
- **대상 파일**: `index.html` (영문), `ko/index.html` (국문)

## 2. 상세 작업 내역

### A. 온도 기호(`℃`) 표준화 및 정렬 개선

- **문제**: 기호와 텍스트(C)의 정렬이 맞지 않고, 브라우저마다 다르게 보이는 문제 발생.
- **해결**:
  - `℃`, `°C` 텍스트를 모두 `<span class="deg">&deg;</span>C` HTML 코드로 변경.
  - CSS 클래스 `.deg`를 추가하여 수직 정렬(`vertical-align: top`) 및 크기 조정.
- **결과**: 모든 제품 사양에서 온도 표시가 일관되고 깔끔하게 출력됨.

### B. 글로벌 고객사 협력 수치 수정

- **변경 사항**: `100여 개` → **`30여 개`**
- **적용**: 국문 및 영문("over 30") 사이트 모두 업데이트.

### C. 영문 내비게이션 메뉴 텍스트 수정

- **변경 사항**: 대문자 표기(`ABOUT US`)를 Title Case(`About us`)로 변경하여 가독성 개선.
- **적용**: `index.html` 상단 메뉴.

### D. 제품 상세 팝업(모달) 텍스트 렌더링 오류 수정

- **문제**: 제품 설명에 포함된 HTML 태그(`<span... >`)가 적용되지 않고 코드로 그대로 노출됨.
- **원인**: JavaScript에서 `textContent` 속성을 사용하여 텍스트를 삽입했기 때문.
- **해결**: `textContent`를 `innerHTML`로 변경하여 HTML 태그가 해석되도록 수정.
