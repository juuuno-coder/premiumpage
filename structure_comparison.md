## 2. Interactive Navigation Logic

Use the top path to navigate upwards.

- **Click `VAISALA`**: Navigates to the main Vaisala product list (defaults to Humidity).
- **Click `Humidity`**: Navigates to the Humidity Category overview.
- **Click `Module/Probe`**: Navigates to the Humidity Category with the "Module/Probe" tab active.
- **Click `HMP1-9`**: (Current Page) Shows the product detail.
